# Documentation

## Identity
This directory contains the central repository of Design Documents, CUJs (Critical User Journeys), and Test Plans for One Human Corp.

## Architecture
Markdown files organized by feature and epic. It strictly adheres to the rule that code must exactly reflect the accepted Design Doc and CUJ.

## Quick Start
Read the index or search for the relevant epic to understand the system design before writing code.

## Developer Workflow
Documentation must be written and committed before any engineering task is executed.

## Configuration
No runtime configuration needed.
